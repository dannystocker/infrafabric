TABLEAU 7 : SEUILS DÉCISION RAPIDE (AIDE-MÉMOIRE OPÉRATIONS)
═══════════════════════════════════════════════════════════════════════════════

À chaque commande >5t, tester :

Test 1 - DISTANCE SEUIL
  < 20km ?   ✓ OUI → Navette interne (coût marginal 25-50€)
             ✗ NON → Test 2

Test 2 - CAPACITÉ INTERNE
  Chauffeur 1 ou 2 disponible ?
             ✓ OUI → Réserver immédiatement (0,30€/km)
             ✗ NON → Test 3

Test 3 - REGROUPEMENT POSSIBLE
  Autre commande région <48h/50h ?
             ✓ OUI → Planifier tournée groupée J+2 (gain -40%)
             ✗ NON → Test 4

Test 4 - VOLUME FAIBLE
  Volume < 10t (moins de 2/3 camion) ?
             ✓ OUI → Chercher hub régional + navette (-70%)
             ✗ NON → Affrètement standard (650-950€, J+2)

RÉSULTAT : 70% commandes doivent emprunter Test 1-3 (interne)
           20% commandes empruntent Test 4 (hub régional)
           10% commandes vraiment affrètement externe (>150km, <5t)
