## RÉSUMÉ EXÉCUTIF (2 min)

### Hypothèse testée
> "Livrer au dépôt le plus proche du fournisseur = toujours moins cher"

### Conclusion
**❌ FAUX** - Infirmée par cas réel Émerge (tuiles 20t, 2 dépôts)

### Preuve empirique principale
```
Scénario A (Application proximité) :
  • Livrer direct Méru (80km, 15t) : 650€
  • Livrer direct Gisors (30km, 5t) : 350€
  • TOTAL : 1 000€ (Gisors 5t coûte 70€/t !!!)

Scénario B (Contre-exemple : Hub régional) :

DÉMENTI : Proximité seule est inefficace sans consolidation
```

2. **Regroupement possible** (30%) - délai ≥ 48h
3. **Délai client** (20%) - urgence détermine mode
4. **Distance** (10%) - secondaire

## CALCULS DÉTAILLÉS - 3 SCÉNARIOS

### SCÉNARIO A : Livraison Directe (Hypothèse proximité) ❌

**Commande :** Émerge tuiles 20t (15t Méru 80km + 5t Gisors 30km)

```
Trajet Méru :
  Affrètement 15t sur 80km : 6,50€/km × 80km = 520€
  Déchargement/services : +130€ (frais Médiafret)
  Subtotal : 650€
  Coût/tonne : 650 ÷ 15 = 43,33€/t

Trajet Gisors :
  Affrètement 5t sur 30km : petit volume surtaxe +50% surcharge
  Base : 6,50€/km × 30 = 195€
  Surcharge petit vol : +155€
  Subtotal : 350€
  Coût/tonne : 350 ÷ 5 = 70€/t ← PÉNALITÉ SOUS-UTILISATION

TOTAL A : 1 000€
Coût/tonne moyen : 1 000 ÷ 20 = 50€/t
Délai : J+2
Taux service : 100%
Problème : Gisors 70€/t vs Méru 43€/t (asymétrie volume)
```

### SCÉNARIO B : Hub Régional (Contre-exemple) ✓ MEILLEUR

**Stratégie :** Livrer Gisors hub (30km proximité), transbordementer, navette Méru

```
Trajet 1 : Fournisseur → Gisors (30km, 20t PLEIN)
  Chauffeur interne fixe : 100€
  Carburant 30km × 0,30€/km : 9€
  Manutention (1h) : 24€
  Subtotal : 133€

Opérations Gisors (transbordement interne) :
  Déchargement 5t : 15€
  Nettoyage bac : 10€
  Recharge 15t : 10€
  Subtotal : 35€

Stockage 12h (15 palettes) : 20€

Trajet 2 : Gisors → Méru (50km, 15t) NAVETTE MARGINAL
  Chauffeur marginal (navette 2×/semaine existante) : 0€
  Carburant 50km × 0,50€/km : 25€
  Subtotal : 25€

TOTAL B : 133 + 35 + 20 + 25 = 213€
Coût/tonne moyen : 213 ÷ 20 = 10,65€/t

ÉCONOMIE vs A : 1 000 - 213 = 787€ (-78,7%)
Délai : J+2 (1 jour transbordement)
Taux service : 99% (excelent, interne)
Avantage : Gisors 5t coûte 10,65€/t (était 70€/t) ← DÉMENTI PROXIMITÉ
```

```
Allocation Émerge (50% équitable) : 147,50€ ≈ 261€
Coût/tonne ÉMERGE : 261 ÷ 20 = 6,87€/t
```

## CONTRE-EXEMPLES DÉCOUVERTS

### Contre-exemple 1 : Volume Asymétrique Prime
- **Méru :** 80km (LOIN) × 15t (75% volume) = 43,33€/t
- **Gisors :** 30km (PROCHE) × 5t (25% volume) = 70€/t
- **Conclusion :** Proximité prédisait Gisors moins cher. Réalité : **62% PLUS cher** car petit volume surtaxé
- **Leçon :** Volume > Distance dans arbre décision

### Contre-exemple 2 : Délai Client Différencié
- **Méru urgent (J+1) :** Affrètement direct obligatoire, pas de regroupement possible
- **Gisors flexible (J+4) :** Regroupement semaine +1 possible = coûts -40%
- **Même région, fournisseur identique, stratégies différentes**
- **Leçon :** Urgence > Proximité économique

### Contre-exemple 3 : Navette Marginal Écrase Distance
- Navette interne : 0,50€/km (chauffeur + carburant seulement, déjà roulée)
- Affrètement : 6,50€/km (13× plus cher)
- **Même sur 100km, regroupement + navette < affrètement 30km**
- **Leçon :** Navette = optimisation gratuite si possible

## FORMULE DE DÉCISION APPLIQUÉE

**À chaque commande >5t, tester dans l'ordre :**

```
TEST 1 - DISTANCE SEUIL
  Si D < 20km ?
    ✓ OUI → Navette interne (FIN, coût 1-3€/t)
    ✗ NON → Continuer

TEST 2 - VOLUME UTILISATION
  Si V/20t > 0,7 (plus de 14 tonnes) ?
    ✓ OUI → Volume dense, continuer TEST 3
    ✗ NON → Volume faible, continuer TEST 4

TEST 3 - DÉLAI URGENCE (si volume dense >14t)
  Si délai < 48h ?
    ✓ OUI → Chauffeur direct urgent (FIN, 10-30€/t)
    ✗ NON → Chauffeur direct optimisé (FIN, 10-20€/t)

TEST 4 - REGROUPEMENT POSSIBLE (si volume faible <14t)
  Si ≥2 autres commandes région < délai+48h ?
    ✓ OUI → REGROUPEMENT (FIN, 6-10€/t) ⭐ OPTIMAL
    ✗ NON → HUB RÉGIONAL (FIN, 10-15€/t) ✓ BON

RÉSULTAT : 70% cmd navette/chauffeur interne
           20% cmd regroupement/hub
           10% cmd affrètement (vraiment inévitable)
```

## SEUIL CRITIQUE DÉCOUVERT

**Indice Proximité Économique :**

```
INDICE = (Volume tonnes / Distance km) × coefficient_urgence

SEUIL : Si INDICE > 0,15 ET délai < 48h
        → Affrètement direct acceptable

       Si INDICE ≤ 0,15 OU délai ≥ 48h
        → Regroupement/Hub recommandé

Cas Émerge :
  Méru : (15 / 80) × 1 = 0,1875 → À limite, regroupement profitable
  Gisors : (5 / 30) × 1 = 0,1667 → Hub INDISPENSABLE
```

## VALIDATION LOCKIENNE (EMPIRISME)

**Principe :** "Rien dans l'esprit qui ne soit d'abord dans les sens" (Locke, 1689)

**Rejet de l'intuition naïve :**
- Intuition : "Gisors 30km PROCHE → moins cher que Méru 80km"
- Mesure : Gisors 70€/t vs Méru 43€/t
- Résultat : Intuition FAUX, contraire confirmé

**Chaîne empiriste appliquée :**
```
1. Hypothèse posée (non testée) : proximité = moins cher
2. Cas réel observé : Émerge 20t
3. Coûts mesurés : 1000€ direct, 213€ hub
4. Écart significatif : 787€ (-78,7%) >> erreur ±5%
5. Cause identifiée : volume asymétrique + transbordement
6. Logique inversée testée : Scénario B (hub régional)
7. Conclusion révisée : Proximité ≠ indicateur dominant
```

**Robustesse :** Écart économique dépasse marge mesure → Conclusion fiable

## QUESTIONS FRÉQUENTES (FAQ)

### Q1 : Comment utiliser ces documents au quotidien ?
**R :** Imprimer AIDE_DECISION Outil 1 & 6 (questionnaire + seuils). À chaque CMD >5t, remplir Outil 1 checklist (3 min). Résultat oriente mode transport.

### Q2 : Et si client refuse délai J+3 (regroupement) ?
**R :** Proposer "livraison J+2 hub régional" (coûts -70%) ou "J+1 chauffeur direct" (coûts -30% vs affrètement). Jamais affrètement directsans négociation.

### Q3 : Qu'en est-il des autres régions (provinces, loin) ?
**R :** Modèle Île-de-France (50-150km). Pour >150km, affrètement devient nécessaire (temps chauffeur trop long). Adapter seuils par région si distances différentes.

### Q4 : Comment intégrer en WMS ?
**R :** Moyen terme : Règles de routage automatisé sur V/D et délai. Court terme : Manuelle avec AIDE_DECISION Outil 4 checklist.

### Q5 : Retard regroupement = coût opportunité ?
**R :** Possible. Comparer : délai +24h vs économie -70% (261€ vs 1000€). Pour client pénalité < 30€ retard, regroupement profitable.
