## Freelancer Classification Test (RDL 2/2015 vs Ley 20/2007)

### RDL 2/2015 Indicators (EMPLOYEE)
- Direct supervision/control
- Fixed working hours
- Integration in client organization
- Client provides tools/materials
- Work performed on client premises
- Exclusive relationship expected

### Ley 20/2007 Indicators (AUTONOMOUS WORKER)
- Independent execution of work
- Own tools/resources
- Multiple clients permitted
- Risk/profit sharing with client
- Flexibility in time/location
- Professional independence

## Essential Clauses to Analyze

1. **Copyright/IP Ownership** → RDL 1/1996 (default is creator; must transfer explicitly)
2. **Confidentiality Terms** → Ley 1/2019 (max 3-year enforcement)
3. **Non-Compete** → Ley 3/1991 + Código Civil (must be reasonable/proportionate)
4. **Tax Responsibility** → Ley 37/1992 + Ley 35/2006 (who pays IVA/IRPF)
5. **Payment Terms** → Ley 3/2004 + Código Comercio (default 30 days + interest)
6. **Termination** → RDL 2/2015 for employees; Código Civil for autonomos
7. **Data Protection** → LOPDGDD + Ley 37/1992 (privacy obligation)
8. **Accounting Records** → RD 1514/2007 + Código Comercio (3-year retention)

## Quick Statute Impact Checklist

- [ ] Is freelancer an employee or autonomous? → Check RDL 2/2015
- [ ] Who owns work product? → Check RDL 1/1996 (copyright default: creator)
- [ ] Are confidentiality terms enforceable? → Check Ley 1/2019 (3-year max)
- [ ] Is non-compete valid? → Check Ley 3/1991 + Código Civil (must be reasonable)
- [ ] Who is responsible for taxes? → Check Ley 37/1992 + Ley 35/2006 (invoicing, IVA)
- [ ] What are payment obligations? → Check Ley 3/2004 (default 30-day net)
- [ ] Data privacy compliance? → Check LOPDGDD + Ley 37/1992 (GDPR+)
- [ ] Accounting requirements? → Check RD 1514/2007 + Código Comercio (3-year retention)
