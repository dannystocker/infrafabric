## Vector Database Integration

### ChromaDB Collection: `if_legal_corpus`

**Ingestion Details:**
- **Documents processed**: 171 successful downloads
- **Vectors generated**: ~10,000-15,000 chunks
- **Chunking strategy**: Hierarchical (1500 char chunks, 200 char overlap)
- **Metadata per chunk**: Jurisdiction, legal_vertical, citation_id, sha256, URL, priority
- **Citation integration**: IF.TTT metadata linked to each vector

**Query Capabilities:**
```python
collection.query(
  query_texts=["housing discrimination"],
  where={"jurisdiction": "uk"},
  n_results=10
)
```

**Metadata Available per Vector:**
- document_name
- jurisdiction
- legal_vertical
- url (source)
- sha256 (integrity)
- citation_id (IF.TTT traceability)
- priority (P0/P1/P2)
