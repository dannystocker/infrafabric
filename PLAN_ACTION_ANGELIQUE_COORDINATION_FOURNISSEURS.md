## PLAN D'ACTION À 90 JOURS - TROIS PHASES

### **PHASE 1 : Formalisation Garantie Délai (JANVIER 2025)**

**Objectif :** Passer de "on essaie de livrer vite" à "on garantit 2-4 jours"

**Actions Concrètes :**

1. **Créer Matrice Délai par Type de Commande** (cette semaine)
   ```
   TYPE              DÉLAI GARANTI   DÉLAI MOYENNE   EXCEPTION
   Chantier <5t      2 jours         2,3j            Si retard fournisseur anticipé
   Chantier 5-20t    3 jours         3,1j            Si retard fournisseur anticipé
   Chantier >20t     4 jours         4,0j            Selon capacité camion
   Retrait magasin   1 jour          1,2j            Direct stock
   ```
   - À faire : Analyser 500 dernières commandes par dépôt/type = extraire délais réels

2. **Valider Capacités avec Chaque Fournisseur** (semaine 2)
   - Appels directs : "Garantissez-vous délai de 48h après commande ?"
   - Fournisseurs prioritaires : Ediliens (tuiles), Ciment (Ciments Calcia), Acier, Bois
   - Résultat : Lister qui peut supporter garantie 48h = 60-70% produits sûrs

3. **Documenter Exceptions Clairement** (semaine 2-3)
   - Si Ediliens dit "non, je peux pas 48h régulièrement"
   - Solution 1 : Augmenter stock de sécurité chez Ediliens (coût)
   - Solution 2 : Proposer au client retard prévisible + remise (-5%)
   - Solution 3 : Communiquer dès jour 1 commande si risque

4. **Publier SLA sur Site Gedimat** (semaine 3)
   ```
   "Gedimat s'engage :
   • Livraison chantier gros volumes 2-4 jours garantis
   • Si retard > délai prévu, remise 10% commande suivante
   • SMS alerte si délai en risque (dès jour 1)"
   ```
   - Impact : Clarifier auprès 500+ adhérents = confiance

---

### **PHASE 2 : Mise en Place Alertes SMS Proactives (FÉVRIER 2025)**

**Objectif :** Quand vous savez qu'une commande retarde, le CLIENT le sait aussi (avant Leroy Merlin/Point P le découvrent)

**Actions Concrètes :**

1. **Intégration SMS API Simple** (2-3 jours de dev)
   - Prestataire : OVH/Twilio/Orange SMS (200-300€/mois)
   - Test 1 : Envoyer SMS manuel à 5 clients test cette semaine
   - Message modèle :
     ```
     "Gedimat : Votre commande tuiles (ref#12345)
     retardée de 48h (usine débordée).
     VOTRE CHOIX :
     A) Attendre + remise 5%
     B) Produit équivalent -5 jours
     Confirmez SMS/Call : 06XX"
     ```

2. **Identifier Commandes à Risque Quotidiennement** (automatisable)
   - Chaque matin : vérifier ACR (accusés réception) fournisseurs vs dates prévues
   - Si écart > 24h = FLAG "risque retard"
   - Vous envoyez SMS client même jour = **client gère planning, pas surprise le jour J**

3. **Mesurer Impact SMS** (KPI n°1)
   - Métrique : "% annulations/transferts chez concurrents avant retard vs après SMS"
   - Cible : Si 100 clients auraient quitté Gedimat faute commu, en retenir 25-35 via SMS = **+5-10M€ CA sauvé**

---

### **PHASE 3 : Commandes Cycliques Prédictives (MARS 2025)**

**Objectif :** Anticiper besoins artisans avant qu'ils commandent = transport mutualisé, délai raccourci, remise garantie

**Actions Concrètes :**

1. **Identifier Clients Cycliques** (immédiat)
   - Analyser ERP Gedimat : qui commande sable/ciment/vis TOUS LES MARDIS ?
   - Cible : 50-100 artisans/PME "hyper réguliers"
   - Exemple : "Maçon Paul à Paris commande 2t ciment + 1t sable même jour chaque semaine depuis 3 ans"

2. **Proposer Partenariat "Commande Cyclique Garantie"** (semaine 1)
   - Pitch : "Vous vous engagez volumes / nous garantissons délai 48h et tarif réduit 5-8%"
   - Avantage Paul : coûts stables, délai garanti, pas surprise = +3% marges Paul
   - Avantage Gedimat : consolidation enlèvements fournisseur, prédiction rotation = -2% coûts logistique

3. **Formaliser Accord Écrit Simple** (template 1 page)
   ```
   ACCORD COMMANDE CYCLIQUE GEDIMAT
   Artisan : [Nom]
   Produits : Ciment sac 35kg, Sable 0-4, Carrelage
   Fréquence : Chaque mardi 10h
   Quantités : [Volumes moyens]
   Délai garanti : Jeudi avant midi
   Tarif : [Prix spécial -6%] valable 12 mois
   Avantage supplémentaire : SMS gratuit retard + remise supplémentaire
   ```

4. **Mettre à Disposition Fournisseur Calendrier Consolidé** (semaine 2-3)
   - Transmettre à Ediliens/Ciments Calcia : "Les mardis on enlève systématiquement Y tonnes"
   - Fournisseur peut optimiser production = qualité, délai, possibilité réduction prix
   - **Win-win : vous savez 7 jours à l'avance → enlèvement optimisé**

## VOS LEVIERS DIRECTS (CE QUE VOUS CONTRÔLEZ)

- **Aujourd'hui :** Vous recevez des news de retard au dernier moment

- **Coût temps :** 1h/semaine
- **Impact :** 0 annulation commande faute délai incompatible planning client

## SCRIPT APPELS FOURNISSEURS (SEMAINE 1)

**Tonalité :** Professionnel, intéressé (pas blâmant), gagnant-gagnant

```
"Bonjour [Fournisseur],
Gedimat veut renforcer notre engagement délai auprès artisans.
Je dois vérifier : pouvez-vous garantir délai 48h APRÈS réception commande
pour produits [Tuiles/Ciment] ?

Si OUI : on va augmenter volumes partenariat et vous donner calendrier prévision
Pour que vous optimisiez production.

Si NON : on doit trouver avec vous :
- Augmenter stock chez vous (coût pour nous)
- Ou réduire promesse client (perte volume)
- Ou mixer : 70% garantis 48h, 30% 72h ?

Quel préférez-vous ?"
```
