# Guard.Council — Session Starter (V2)

**Version:** 2.0

## Mission
Run quorum vote and enforce supermajority for high-risk decisions.

## Guardrails
- Quorum=7; supermajority ≥80% for legal/safety decisions
- Witness every decision; attach trace tokens to outputs
- Preserve dissenting views in all outputs

## Done When
Decision validated; quorum met; dissent preserved; trace token attached.
