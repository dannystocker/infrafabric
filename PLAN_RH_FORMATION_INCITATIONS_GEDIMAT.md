## 1. GESTION ÉQUITABLE DE LA CHARGE

### 1.1 Chauffeurs PL Internes : Équilibrage Heures/Km/Tonnage

**Diagnostic actuel**
- 2 chauffeurs internes font l'ensemble des navettes inter-dépôts
- Aucune mesure formelle du temps/km/tonnage par chauffeur
- Risque : "Même chauffeur toujours sur urgences" → burnout + surcharges injustes

**Mise en place équitable (Court terme : Mois 1)**

| Mesure | Mécanique | Cible | Fréquence |
|--------|-----------|-------|-----------|
| **Rotation missions urgentes** | Planifier semaine N : chauffeur A urgences, N+1 chauffeur B | Chacun ≤2 urgences/semaine | Hebdomadaire |
| **Équilibrage kilométrage** | Tracker km réels par trajet (GPS), total/mois par chauffeur | ±10% équilibre (ex : 2400 km vs 2600 km) | Mensuel |
| **Tonnage moyen par tournée** | Tonnage livré / nb tournées par chauffeur/mois | Équilibré ±15% | Mensuel |
| **Rémunération transparente** | Rémunération fixe salaire + prime équité (si km/urgence déséquilibrés, compensation) | Prime 200-300€/mois si injustice | Trimestrial |

**Outil de suivi** (Excel simple + Google Sheets collaborative)
```
Semaine 1 (N) :
Lundi  : Chauffeur A → Navette Méru/Lieu (25km, 8t)
Mercredi: Chauffeur B → Urgence Gisors (35km, 5t)
Vendredi: Chauffeur A → Navette Gisors/Breuilpont (30km, 6t)

Mois : A = 120km + 19t (urgent : 1)  /  B = 105km + 18t (urgent : 2)
→ Rééquilibrer mois N+1 (B moins urgent, A une urgence plus)
```

### 1.2 Coordinateurs Dépôts : Distribution Commandes Multi-Dépôts sans Favoritisme

**Diagnostic actuel**
- 3 responsables dépôts (Méru, Gisors, Lieu) défendent "leur" dépôt
- Exemple : Méru avec 15t demande livraison directe vs Gisors (5t, plus près) avec urgence
- Aucune règle écrite → chacun négocie → Angélique arbitre manuellement

**Mise en place équitable (Court terme : Mois 1-2)**

| Responsable Dépôt | Indicateur Équité | Mesure | Cible Mensuelle |
|-------------------|-------------------|--------|-----------------|
| **Priorité urgence clients** | % urgences client + tôt traitées par dépôt | Urgent(8+/10) = priorité absolue indépendant dépôt | ≥95% urgences délai respecté |
| **Distribution juste des navettes** | Palettes/navette par dépôt vs volume stock | Pas 1 dépôt toujours "relais" (dernier reçoit) | Rotation : chacun reçoit "premier" 1 semaine/4 |
| **Commandes reportées injustement** | Nombre fois où dépôt demande = reporté sans justification | Justification écrite si report (retard fournisseur, surcharge logistique) | 0 report injustifié/mois |
| **Engagement collaborative score** | Dépôt accepte livraison "relais" même si moins optimal | Bonus pointage si dépôt accepte flexibilité | Voir section Incitations |

**Processus décisionnel écrit (Affichage dépôts)**
```
RÈGLE ARBITRAGE MULTI-DÉPÔTS GEDIMAT
1. URGENCE CLIENT prime (score ≥7/10) → Dépôt concerné reçoit livraison directe
2. PROXIMITÉ FOURNISSEUR second (économie transport) → Plus proche route
3. VOLUME DÉPÔT tertaire (justifie pas coût additionnel seul)

Exemple Emeris 15t+5t :
  Méru : 15t (pas urgent), Gisors : 5t (urgent chantier lundi)
  → Gisors prioritaire (urgence) MÊME SI 5t < 15t
  → Économie transport possible ? (Emeris plus près Gisors ?)
    OUI → Livrer Gisors, navette Méru ✓
  → Coûts navette acceptable ? <20% coût additionnel ?
    OUI → Décision final GISORS
```

## 2. FORMATION COORDINATION COLLABORATIVE

### 2.1 Programme Formation Angélique + 3 Responsables Dépôts (2 heures)

**Objectif pédagogique**
Passer de "défense territoriale intuitive" à "arbitrage collaboratif intelligent basé data"

**Format**
- 1 session 2h groupe (Angélique + 3 resp. dépôts)
- Jour : Mardi après-midi (moins chargé logistiquement)
- Lieu : Salle réunion Lieu (neutre, pas favoriser dépôt)
- Animateur : Manager opérationnel (Dan) + support Angélique

**Contenu Détaillé (120 minutes)**

| Durée | Module | Contenu | Livrables |
|-------|--------|---------|-----------|
| **20 min** | **Contexte financial** | Coûts réels : affrètement 300-400€, navette 80-100€, surcoûts urgence 30-50% | Diapo coûts cachés comparatif 3 scénarios Emeris |
| **20 min** | **Cas réel Emeris** | Simulation : 15t Méru (pas urgent) + 5t Gisors (urgent chantier). Q : quoi prioritaire ? Pourquoi ? | Tableau décision rempli groupe, discussion résultats |
| **25 min** | **Scoring multicritère** | Grille 4 critères : urgence (40%), coût transport (35%), proximité (15%), volume (10%). Calculer score 0-10. | Feuille scoring + 2 cas pratiques complétés |
| **20 min** | **Scripts communication clients** | Cas 1 : Client fâché retard (gestion objection "pourquoi pas prioritaire ?"). Cas 2 : Communication proactive urgence. | 2 scripts à emporter, simulations orales |
| **10 min** | **Dashboard + CRM** | Tour rapide : où noter urgence_score, où voir coûts transport, où documenter décision. | 1 page mode d'emploi dashboards |
| **5 min** | **Q&A + engagement** | Chacun dit 1 inquiétude + 1 point positif du nouveau system | Notes RH (voir gestion changement) |

**Supports pédagogiques à préparer**
1. **Cas Emeris Tuiles** (fiche 1 page) : Contexte + données + 3 options + scores calculés
2. **Grille Scoring** (affiche laminée) : Urgence/Coût/Proximité/Volume avec exemples concrets
3. **Scripts clients** (fiches recto-verso) : "Retard livraison" / "Communication proactive urgence"
4. **Dashboard mockup** (imprimé) : Où noter urgence_score, voir coûts, valider décision
5. **Feuille participant** (bloc notes) : Simulations à remplir pendant formation

**Durée 2h suffisante ?**
✓ OUI (45 min théorie légère + cas pratique + scripts + tool tour)
✗ NON si : viser certification complète, tests d'acquis rigoureux
→ **Recommandation** : +30 min suivi opérationnel sur 2-3 cas réels (semaines 1-4 post-formation)

### 2.2 Ateliers Arbitrage Collaboratif (Récurrent)

**Format mensuel (30 min, 3ème mercredi du mois)**
- Revoir 3-5 cas complexes du mois (urgence vs volume, débats dépôts)
- Analyser : décision prise était-elle optimale ? Quoi appris ?
- Renforcer collaboration ("tu as raison sur coûts, j'ai raison sur urgence → solution ensemble")

**Cas typiques à simuler en atelier**
```
Cas 1 : Fournisseur retard + urgence client
        → Action proactive Angélique ? Escalade quand ?
        → Scripts escalade fournisseur

Cas 2 : Dépôt refuse être "relais"
        → Négociation Angélique + resp. dépôt
        → Justification coûts si nécessaire

Cas 3 : Consolidation vs urgence mix
        → 2 urgences contradictoires (Méru+Gisors)
        → Attendre consolidation ou 2 enlèvements ?
```

## 3. SYSTÈME INCITATIONS COLLABORATION INTER-DÉPÔTS

### 3.1 Bonus Satisfaction Globale (vs Dépôt Individuel)

**Diagnostic problème**
- Responsables dépôts optimisent "leur" dépôt seul (coûts locaux, urgences locales)
- Pas d'incitation reconnaître besoin autres dépôts
- Résultat : défense territoriale, arbitrages conflictuels

**Nouvelle structure bonus (Annuel)**

| Structure Bonus | Calcul | Cible | Impact RH |
|-----------------|--------|-------|-----------|
| **NPS Dépôt** (20% poids) | (Clients très satisfaits - clients insatisfaits) / Total clients dépôt | ≥50 (secteur BTP) | Reconnait satisfaction clients |
| **Taux collaboration** (40% poids) | (Accepte livraison relais + délais flexibles) / (Total demandes service) × 100 | ≥80% | Prémie collaboration |
| **Coût transport/palette** (30% poids) | Coût total transport dépôt / nb palettes manipulées | ≤8€/palette (vs 15€ affrètement) | Prémie efficacité |
| **Absence incidents** (10% poids) | Retards causés par dépôt / Total livraisons | =0% (cible) | Qualité opérationnelle |

**Formule Bonus**
```
Bonus Responsable Dépôt = 2 500€ base × coeff_performance

Où coeff_performance = (NPS_score×0.2) + (Collaboration_score×0.4) +
                       (Transport_score×0.3) + (Incidents_score×0.1)

Exemple Responsable Méru :
  NPS : 50/100 = 0.5/1 × 0.2 = 0.10
  Collaboration : 85% = 0.85/1 × 0.4 = 0.34
  Transport : 7.5€/pal = (8-7.5)/8 × 1 × 0.3 = 0.01875 ≈ 0.02
  Incidents : 1 retard = 0.8/1 × 0.1 = 0.08
  ─────────────────────────────────
  coeff = 0.10 + 0.34 + 0.02 + 0.08 = 0.54

Bonus = 2 500€ × 0.54 = 1 350€ (vs potentiel max 2 500€ si coeff=1.0)
```

**Benchmark comparatif (Incitations avant vs après)**

| Scenario | Bonus Ancien | Bonus Nouveau | Delta | Incitation |
|----------|--------------|---------------|-------|-----------|
| Méru : Volume max, refuse relais, client fâché | 1 000€ | 800€ | -200€ | ✗ Défense territoriale pénalisée |
| Gisors : Accepte relais, aide autres, satisfaction ok | 1 200€ | 1 800€ | +600€ | ✓ Collaboration récompensée |
| Lieu : Balanced, coûts bas, incident | 1 400€ | 1 700€ | +300€ | ✓ Efficacité reconnue |

### 3.2 Bonus Satisfaction GLOBALE (3 dépôts) vs Dépôt Individuel

**Ratio proposé**
- 70% Bonus basé NPS GLOBAL (3 dépôts ensemble) → Prémie collaboration
- 30% Bonus basé performance dépôt individuel → Maintient qualité locale

**Formule Mixte**
```
Bonus = [NPS_Global × 0.35 + Collaboration × 0.35] × 2 500€ (70% part commune)
      + [Transport_cost × 0.20 + Incidents × 0.10] × 750€ (30% part individuelle)

Exemple : Si NPS_Global = 55/100
  Tous 3 resp. dépôts reçoivent : (55/100 × 0.35 + 85/100 × 0.35) × 2 500
                                = 1 625€ (BASE commune)
  + Selon perfs individu : 800€ (Méru) / 1 050€ (Gisors) / 950€ (Lieu)

  Total : Méru 2 425€ / Gisors 2 675€ / Lieu 2 575€

  → Gisors récompensé pour collaboration (NPS global bon + son engagement)
```

**Avantage du ratio 70/30**
- ✓ 70% global : force collaboration ("si l'un souffre, tous souffrent")
- ✓ 30% individuel : garantit pas sandbagging (pas laisser dépôt partir à dérive)
- ✓ Transparent : calcul simple, auditable mensuellement

### 3.3 Reconnaissance Coordination Efficace (Économies Transport Mesurées)

**Mécanisme bonus Angélique**
```
Bonus Angélique = Base 1 500€ + Prime Coordination

Prime Coordination = Économies_mesurées × Coeff_complexité

Économies mesurées = (Coûts_before_process - Coûts_after_process) × 0.5 partage
(Gedimat reprend 50% économies, 50% réinvesti en formation/outils)

Complexité cases traités = (Urgences 8+/10 arbitrées) × 200€
                          + (Consolidations multi-dépôts) × 150€
                          + (Escalades fournisseur réussies) × 100€

Exemple mois 1 (après formation) :
  Économies Emeris (consolid) : 150€ × 0.5 = 75€
  Économies autres (5 cas) : 1 200€ × 0.5 = 600€
  Cases urgences traitées : 8 × 200€ = 1 600€
  Consolidations : 3 × 150€ = 450€
  Escalades : 2 × 100€ = 200€
  ──────────────────────────
  Prime = (75+600) + (1600+450+200) = 2 925€

Bonus Angélique = 1 500 + 2 925 = 4 425€
(vs salaire ~2 800€/mois → prime 150% salaire reconnaît criticité rôle)
```
