1. **Signal Capture (IF.trace):** Observable patterns requiring investigation
2. **Primary Analysis (ChatGPT-5):** Rapid multi-perspective breakdown
3. **Rigor & Refinement (Human):** Stress-test AI outputs, verify sources
4. **External Research (Stage 4):** Peer-reviewed empirical grounding
5. **Framework Mapping:** Patterns → reusable IF components
