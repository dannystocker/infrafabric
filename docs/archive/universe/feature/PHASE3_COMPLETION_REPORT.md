### 2. Approval Thresholds

- **100%:** Constitutional amendments, existential decisions
- **90%+:** Major framework changes, >$100K resource allocation
- **75%+:** Standard decisions, regular case approvals
- **70%+:** Minor decisions, process adjustments
- **<50%:** Rejection, returns for revision

### 3. Voting Formula

```
weighted_approval = (
  core_guardians_avg * 0.50 +
  western_philosophers_avg * 0.20 +
  eastern_philosophers_avg * 0.20 +
  if_ceo_avg * 0.10
)
```

### 4. Special Mechanisms

**Contrarian Guardian Veto:**
- Triggered when approval exceeds 95%
- 2-week cooling-off period mandated
- Prevents groupthink on near-unanimous votes

**Appeal Process:**
- Concerned parties can challenge decisions
- Contrarian Guardian reviews appeals
- Violations trigger re-vote

**Constitutional Amendments:**
- Requires 3 Council members
- 2-week public comment period
- Requires 100% unanimous vote
