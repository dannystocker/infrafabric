## Council Composition & Voting

### The 21 Voices (Current)

**Core Guardians (7 seats, 50% weight):**
- Technical Guardian (1.2 weight)
- Ethical Guardian (1.1 weight)
- Legal Guardian (1.0 weight)
- Scientific Guardian (1.3 weight) - HIGHEST WEIGHT
- Business Guardian (1.1 weight)
- Coordination Guardian (1.0 weight)
- Rory Sutherland - Behavioral Alchemy (1.0 weight) - Added 2025-11-18

**Western Philosophers (3 seats, 20% weight):**
- Empiricist (Locke tradition) - Observable evidence
- Rationalist (Descartes tradition) - Logical coherence
- Pragmatist (Peirce tradition) - Practical consequences

**Eastern Philosophers (3 seats, 20% weight):**
- Buddhist - Interdependence
- Daoist - Balance & harmony
- Confucian - Social harmony

**CEO Council (8 facets, 10% weight):**
- Light Side: Idealism, Pragmatism, Vision, Realism
- Dark Side: Strategy, Ruthlessness, Ambition, Pragmatic-Dark

**Contrarian Guardian (1 seat, veto power):**
- Devil's Advocate - Can veto decisions >95% approval
- Triggers 2-week cooling-off period
- No formal vote weight (special mechanism)

### Voting Formula
```
weighted_approval = (
  core_guardians_avg * 0.50 +
  western_philosophers_avg * 0.20 +
  eastern_philosophers_avg * 0.20 +
  if_ceo_avg * 0.10
)
```

### Approval Thresholds
- 100%: Constitutional amendments, existential decisions
- 90%+: Major framework changes, large resource allocation
- 75%+: Standard decisions (default)
- 70%+: Minor decisions, process adjustments
- <50%: Rejection (proposal returns for revision)

### Quorum Requirements
- Minimum: 15 of 21 voices (71.4%)
- Core Guardians: At least 5 of 7
- Philosophers: At least 4 of 6

## Navigation Tips

1. **Top-down (systematic):**
   1. Read PHASE3_COMPLETION_REPORT.md
   2. Read IF_CONSTITUTION.md (Intro + Articles I-II)
   3. Browse ethics/ directory
   4. Read specific council decisions as needed

2. **Bottom-up (detailed):**
   1. Pick a guardian profile from ethics/
   2. Look at their voting record (infrafabric-IF-annexes.md)
   3. Read the related council debate
   4. Check IF_CONSTITUTION.md for how it fits

3. **Decision-focused:**
   1. Check Council Decision Timeline (above)
   2. Read the specific decision file
   3. Look at related votes
   4. Check Constitution for decision framework

4. **Quick lookup:**
   1. Need guardian info? → ethics/ directory
   2. Need voting details? → GUARDIANS_PANEL.yaml
   3. Need laws? → IF_CONSTITUTION.md
   4. Need history? → infrafabric-IF-annexes.md
