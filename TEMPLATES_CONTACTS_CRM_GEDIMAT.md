## INTRODUCTION - UTILISATION DES TEMPLATES

### Objectif
Structurer et documenter **complètement** tous les contacts clés pour :
- Pérenniser le relationnel si Angélique absente/départ
- Créer redondance (2+ contacts par fournisseur)
- Tracabiliser accords informels (délais, tarifs, flexibilité)

### Comment utiliser
1. **Copier le template** dans Google Sheets / Excel
2. **Remplir les données existantes** (ce qu'Angélique connaît)
3. **Actions manquantes** : Appeler fournisseur pour compléter (contact secondaire)
4. **Partage** : Manager + Angélique (lecture/écriture)
5. **MAJ** : À chaque changement (nouveau contact, nouvel accord)

### Responsabilité
- **Angélique** : Collecte données, remplissage initial
- **Manager** : Validation, suivi compléments manquants
- **Fournisseur** : Confirme accords, contact secondaire
